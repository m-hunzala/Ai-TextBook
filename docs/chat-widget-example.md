## Try It Out

Select any text on this page to see the "Ask about this" button appear. Then click it to ask a question about the selected content. The AI will provide a response based on the documentation with citations to the relevant sources.

For more information about integrating the widget into your own Docusaurus site, check out the [Chat Widget Integration](chat-widget.md) documentation.

```jsx
import ChatWidget from '@site/src/components/ChatWidget';

<ChatWidget 
  apiEndpoint="https://your-rag-api.com" 
  apiKey="your-api-key"
/>
```
