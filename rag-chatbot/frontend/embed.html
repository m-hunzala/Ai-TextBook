Embed Instructions for Book RAG Chatbot

This page demonstrates how to embed the Book RAG Chatbot into your published book web page.

Embed Snippet:

<!-- Book RAG Chatbot Widget -->
<div id="book-rag-widget-container" class="book-rag-widget-container">
  <iframe 
    src="PATH_TO_YOUR_EMBED_HTML" 
    class="book-rag-widget-iframe"
    title="Book Assistant Chat"
    style="border: none;"
    allow="clipboard-write"
  ></iframe>
</div>
<script>
// Optional: Set API URL dynamically
window.BOOK_RAG_API_URL = 'YOUR_API_BASE_URL';
</script>

How to Use:

  1. Replace PATH_TO_YOUR_EMBED_HTML with the path to your embed HTML file
  2. Replace YOUR_API_BASE_URL with your actual API base URL
  3. Include this code snippet in your book's HTML page
